User Guide & Help

What is Work Hour Tracker?

Work Hour Tracker is a self-contained browser app for logging and analyzing your daily work sessions. All data is stored locally in your browser — no account or internet connection required after the page loads.

Interface at a Glance

  • Left Panel — Live Tracker: Your active session controls, live timers, and daily progress bar.
  • Right Panel — Insights / Logbook: Switch between your weekly analytics view and your full shift history using the two tabs at the top of the panel.
  • Footer Toolbar: Access CSV export, clipboard copy, JSON backup/restore, leave management, help, settings, and the factory reset (Wipe).
  • Header Bar: Set your active timezone, configure your daily goal, and switch the app theme.

Quick Start (3 Steps)

  • Step 1 — Set your goal: Before starting, confirm your Daily Goal in the header (default: 8h 0m). Click the field to open goal presets, or type a custom value. The goal for each day is permanently locked the moment you submit your first shift, so set it correctly upfront.
  • Step 2 — Track your day: Click Start Work when you begin. Use Take Break when stepping away and Resume Work to continue. The Production and Break timers update in real time.
  • Step 3 — Submit your shift: Click Submit EOD at the end of your day. A dialog will prompt you to add shift notes before the entry is written to your Logbook permanently.

The Command Center HUD

Press / to open the intelligent Command Palette. It lets you drive the entire application using natural language and smart commands without ever touching your mouse.

Advanced Database Engines

  • Natural Language Write Engine: Type add or logshift followed by a natural sentence to skip the manual forms and instantly construct a backdated database log.
    • Absolute Ranges (12h/24h): 9am to 5pm, 14:00 - 18:00
    • Explicit Durations: 8h 30m, 4.5h, 90m
    • Relative & Absolute Dates: yesterday, today, 05/12, 05/12/2026
    • Collision Warnings: If a log already exists for the parsed date, the UI throws a bright ⚠️ Conflict warning.
    Example: Type add yesterday 9am to 5pm worked on UI architecture. It calculates 8 hours of work time for yesterday, injects the note, and displays a HUD button. Press Enter to instantly commit it.
  • Deep Log Search Engine: Type log followed by any combination of filters (e.g. >5h, last month, over, plus custom text) to extract up to 5 matching database records directly into the HUD.
    Example: Type log meeting >4h last month to find long shifts with "meeting" in the notes from last month. Press Enter to copy it.
  • Interactive Log Quick-Edits: Type edit followed by a keyword to search past shifts. Hit Enter on a result to instantly transform the HUD into a quick-edit text field for modifying notes seamlessly.
  • Dynamic Metrics Aggregation: Type metrics followed by a timeframe (today, week, month, all) to sum up your exact Work and Break hours.
  • NLP Leave Engine: Type leave followed by a type and date to instantly book leave directly into the database.
    Example: Type leave sick tomorrow, leave casual next monday, or leave holiday 12/25/2026 to bypass modal forms completely.

Smart Quirks & Utilities

  • Zero-Type Dashboard: Simply pressing / without typing will contextually show your Live Session Timer card along with smart toggle defaults (like Pause/Resume) depending on whether you are working or resting.
  • Live Theme Auditioning: Search for themes (e.g. theme cobalt) and simply use your Arrow Keys to preview them in real-time across the app before hitting Enter.
  • Smart Goal Calculus: Type goal followed by your desired time (e.g. goal 1h 30m). The UI predictively calculates and displays what percentage of the new goal your current session completes.
  • Dynamic Calculator: Type any math expression to instantly evaluate it.
    Example: Type 150 / 60 and press Enter to instantly copy 2.5 to your clipboard.
  • Fuzzy Search & Aliases: Type acronyms or synonyms.
    Example: Type sw to find "Start Work", or nuke for "Wipe All Data".
  • Quick Shift Notes: Type note followed by text while working.
    Example: Type note Fixed the backend bug. It silently appends this note to your running timer.
  • Web Search: Type g followed by a query.
    Example: Type g css flexbox to instantly open a Google tab.

Keyboard Shortcuts Reference

The application features a rich suite of keyboard hotkeys designed specifically for power users. All active mappings are listed below:

  • Spacebar: Starts a new shift, takes a break, or resumes active work.
  • S: Opens the End of Day (EOD) submission modal.
  • Ctrl + S / Cmd + S: Downloads the backup JSON file and opens an email draft template.
  • Alt + 1 / Alt + 2: Swaps directly between the Insights and Logbook tabs.
  • Slash (/): Opens the intelligent Command Center HUD.
  • Question Mark (?): Opens this Help Guide overlay.
  • Escape: Exits input fields, closes modal overlays, collapses the filter drawer, and deselects highlighted logs.
  • Tab: Navigates fields sequentially with focus traps active inside modals.
  • Alt + K: Opens the Logbook search box and expands the advanced filters automatically.
  • Alt + N: Launches the Add Manual Log modal instantly.
  • Alt + T: Cycles through the four design themes (Onyx Black, Cobalt Glass, etc.).
  • Alt + Plus/Minus (= / -): Fine-tunes your Daily Goal by +/- 15 mins.
  • Alt + Left / Alt + Right: Steps back and forward through the Insights calendar week.
  • Selected Log Actions: Highlight an entry with the arrow keys, then press:
    • E — Edit the selected shift record.
    • C — Copy the shift record cleanly to your clipboard.
    • N or Enter — View shift notes in a popup overlay.
    • D or Backspace / Delete — Trigger the deletion confirmation flow.

Session Controls

  • Start Work: Begins a new session and starts the Production timer. Only available when no session is active.
  • Take Break: Pauses the Production timer and starts the Break timer. Available while working.
  • Resume Work: Stops the Break timer and resumes Production. Replaces the Take Break button while on break.
  • Submit EOD: Ends and saves the current session to your Logbook. You will be prompted to enter shift notes. This action is final — the shift is written to permanent storage.

Progress Bar & ETA

  • The horizontal bar between the two timers shows your progress toward today's daily goal.
  • Left label: Current completion percentage and remaining time.
  • Right label: Estimated end-of-day time in BST based on your current pace.
  • The bar turns fully green once your goal is reached.

Goal Anchor

When you submit your first shift on any given day, the daily goal active at that moment is permanently saved ("anchored") to that entry. This ensures that changing tomorrow's goal does not retroactively alter your historical records.

Best practice: Always verify your Daily Goal in the header before submitting your first shift of the day. If you need to correct a previously anchored goal, go to the Logbook tab, click the Edit icon on that entry, and update the goal value directly.

Intelligent Mode — Smart Tracking

When Intelligent mode is active (configured in Settings), the app runs three protective behaviors automatically:

  • Idle Lock: If your keyboard and mouse are inactive beyond the configured threshold (default: 60 minutes), the session pauses and a dialog appears when you return. You choose whether the idle period counts as work, break, or is discarded entirely.
  • Sleep / Tab Suspension Detection: If your computer sleeps, the browser suspends the tab, or the system freezes, the app detects the gap on wake and prompts you to decide how to handle the missing time — the same as idle recovery.
  • Midnight Rollover: If a session is still active at 12:00 AM, the app automatically submits the current shift for the prior day and starts a clean session for the new date. No data is lost.

Simple mode disables all three behaviors above. The timers run continuously with no interruptions, regardless of inactivity or system sleep.

Smart Time Inputs

Whenever you edit a time value (like in the Add Log or Edit Shift menus), you can use shorthand formats to speed up entry. The app will automatically format it to HH:MM when you press Enter or click away.

  • Numbers: Typing 8 converts to 08:00.
  • Decimals: Typing 8.5 converts to 08:30.
  • Minutes: Typing 45m converts to 00:45.
  • Hours & Minutes: Typing 8h 30m converts to 08:30.

Weekly Progress Overview

The Insights tab (right panel) shows your aggregated weekly performance. Use the left/right arrows to navigate between weeks. Double-click the week label to jump back to the current week.

  • Total Work / Total Break: Cumulative hours for the displayed week.
  • Weekly completion %: Your progress toward the 40-hour weekly target, shown in blue.
  • Status message: A short contextual summary — e.g., whether you are on track, ahead, or behind pace for the week.

M–T–W–T–F Day Segments

The five horizontal bars represent Monday through Friday of the selected week. Each bar fills proportionally based on hours logged against that day's goal.

  • A bar that is fully green means the daily goal was met or exceeded for that day.
  • A bar with a striped pattern indicates a scheduled leave day. Leave days are excluded from pace and goal calculations.
  • The current day's bar updates live as your session progresses.

Shift Badges

Each log entry in the weekly breakdown carries colored badges summarizing that shift:

  • W Work Time — Total productive hours logged.
  • B Break Time — Total break duration.
  • O Overtime — Shift met or exceeded its anchored goal.
  • U Under Goal — Shift did not reach its anchored goal.
  • F Focus % — Ratio of work time to total time (work + break). Higher is more focused.

Pace Required

Displayed in the weekly breakdown, this figure tells you how many hours per day you need to average across the remaining active (non-leave) weekdays to reach your 40-hour weekly target. It recalculates automatically as you log hours each day.

Viewing & Navigating Entries

The Logbook tab (right panel) lists every submitted shift entry in reverse chronological order. Each entry shows the date, login/logout times, work and break duration, daily goal, focus percentage, and any shift notes.

Adding a Manual Entry

Click Add Log in the Logbook header to open the manual entry form. Fill in the following fields:

  • Date: Enter in MM/DD/YY format, or use the natural language shortcuts today or yesterday.
  • Login / Logout Times: The start and end times of the shift.
  • Work Hours / Break Hours: Duration of productive time and break time within the shift.
  • Daily Goal: The target hours for that specific day.
  • Notes: Optional — tasks completed or any relevant context.

Editing & Deleting Entries

  • Edit (pencil icon): Opens the entry in an editable form. You can update any field, including the anchored goal, notes, hours, or date. Save to commit changes.
  • Delete (trash icon): Permanently removes the entry. This action cannot be undone. A confirmation prompt will appear before deletion.

Filters & Saved Views

Click Filters in the Logbook header to expand the filter drawer. Available filter criteria include:

  • Search: Full-text search across shift notes and timestamps.
  • Date Range: Filter to a specific date window using the date picker fields.
  • Work Hours: Show only shifts above or below a specified hour threshold.
  • Overtime / Under Goal: Filter by whether shifts met their daily goal.

Once you have configured a filter combination you use regularly, save it as a named Saved View. Saved views appear in the dropdown at the top of the filter drawer and can be recalled with a single click.

Managing Leaves & Holidays

Click the Leaves icon in the footer toolbar to open the leave manager. Add scheduled leave days (annual leave, sick days, public holidays) by date. Leave days are rendered with a striped pattern in the Insights day segments and are excluded from pace and weekly goal calculations.

Export & Sharing

  • CSV (spreadsheet icon): Downloads a .csv file of all visible log entries. Columns include date, login/logout times, work hours (decimal), break hours, daily goal, focus %, and notes. Suitable for submitting to HR or importing into Excel / Google Sheets.
  • Copy (clipboard icon): Copies your log history to the clipboard as a formatted plain-text email summary. Entries are grouped by day, work/break totals are shown per day, and any shift under 7 hours is flagged. Paste directly into a Gmail draft with Ctrl+V / Cmd+V.

Backup & Restore

  • Save / JSON backup (download icon): Downloads a .json file containing your entire log database and app state. Store this file somewhere safe. All data lives in your browser's local storage — clearing browser data or switching devices will erase it without a backup.
  • Load / JSON restore (upload icon): Imports a previously saved .json backup file. Existing logs are merged with the imported data; duplicate entries are de-duplicated automatically.
  • Recommended cadence: Export a JSON backup at least once per week.

Settings (⚙ icon)

  • Simple / Intelligent mode: Switch between tracking modes. Intelligent mode enables idle lock, sleep detection, and midnight rollover. Simple mode runs timers continuously without any interruption or auto-detection.
  • Enable Idle Lock: Toggle automatic idle detection on or off (Intelligent mode only).
  • Idle Threshold: The slider sets how long you must be inactive (5–120 minutes) before the idle lock triggers. Default is 60 minutes.
  • Auto-save Backup: When enabled, the app automatically downloads a JSON backup file at the end of each day when you submit your EOD shift.
  • Auto-email Summary: When enabled, submitting EOD automatically copies your log summary to the clipboard and opens a new Gmail draft, ready to paste.

Theme & Timezone

  • Theme: Click the theme toggle button in the header to open the theme picker. Choose from SF Dark, SF Light, Onyx Black, or Cobalt Glass. Your selection is saved automatically.
  • Timezone: Click the timezone label (PST/MST/CST/EST) in the header status bar to switch the primary clock timezone. The secondary clock always displays BST as a fixed reference. Your selection persists across sessions.
  • Daily Goal Presets: Click the Daily Goal field to open the preset picker. Use −15m / +15m to fine-tune, or double-click either preset pill to permanently reassign it to a custom value.

Factory Reset

The Wipe button (red trash icon) permanently deletes all logs, saved views, leave entries, and app settings. This action is irreversible. Always export a JSON backup before using this option.
